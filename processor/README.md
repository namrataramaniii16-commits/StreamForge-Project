# StreamForge - Stream Processing Module

## Overview

This module is responsible for implementing the Stream Processing component of the StreamForge project using Faust and Apache Kafka.

## Responsibilities

- Initialize the Faust application
- Connect to Apache Kafka
- Consume truck telemetry events
- Build the stream processing pipeline
